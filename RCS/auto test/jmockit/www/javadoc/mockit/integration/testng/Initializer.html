    public final class Initializer
extends Object
implements org.testng.ITestNGListener
    A test listener implementation for TestNG that will properly initialize JMockit before any tests are executed.

    When using TestNG 6.2 or newer, this class is automatically loaded. With older versions of TestNG (at least 5.14), it must be explicitly specified. One way to configure TestNG to use this class as a listener is to pass "-listener mockit.integration.testng.Initializer" as a command line argument. Another way is through testng.xml configuration. Please check the TestNG documentation for details.
